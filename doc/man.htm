Comment jouer ?

Liste des touches :

Flèches directionnelles Déplacer le joueur, naviguer dans l'inventaire
Espace Utiliser l'objet 1 / Assigner l'objet 1 dans l'inventaire
A Utiliser l'objet 2 / Assigner l'objet 2 dans l'inventaire
Z Utiliser l'objet 3 / Assigner l'objet 3 dans l'inventaire
C Ouvrir la mini-Carte
P Ouvrir/Fermer le menu pause
I Ouvrir/Fermer l'inventaire
Page Haut Mettre le panneau de contrôle en haut
Page Bas Mettre le panneau de contrôle en bas
F1 Prendre une capture d'écran (sauvée dans screenshot/)
S Enregistrer la partie (la dernière sauvegarde est chargée automatiquement)
Échap Quitter

Comment utiliser l'éditeur de cartes ?

Nécessite Python et PyQt4 (sous Linux Debian/Ubuntu, paquet python-qt4)

Cliquez sur Editor.py pour lancer l'éditeur, ou ./Editor.py en console (le chemin de la carte à ouvrir peut être passé en argument de la ligne de commande)

Raccourcis clavier :

Ctrl+O Ouvrir une carte
Ctrl+S Enregistrer la carte
Ctrl+Shit+S Enregistrer la carte sous un autre nom de fichier
Ctrl+Q Quitter l'éditeur
U Annuler le dernier placement de tile
Plus Ajouter une unité
Moins Supprimer une unité
Flèches directionnelles Se déplacer dans les zones
