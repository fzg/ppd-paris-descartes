Actions

  • Déplacer un objet :
    Certains objets peuvent être déplacés en les poussant : placez-vous contre l'objet et continuer à avancer contre lui pendant 1 seconde pour le mettre en mouvement

Configuration

Deux fichiers de configuration sont utilisés :

  • config/config.txt : contient les préférences du jeu (emplacement du panel, musique, ...)
    Vous pouvez supprimer ce fichier si vous souhaitez retrouver la configuration par défaut
  • config/save.txt : contient la sauvegarde de la progression du joueur
    Vous pouvez supprimer ce fichier si vous souhaitez recommencer le jeu à zéro

Comment utiliser l'éditeur de cartes ?

L'éditeur nécessite Python et PyQt4 (sous Linux Debian/Ubuntu, installez le paquet python-qt4)

Cliquez sur Editor.py pour lancer l'éditeur, ou lancez la commande ./Editor.py en console (le chemin de la carte à ouvrir peut être passé en argument de la ligne de commande, par exemple : ./Editor.py ../bin/data/map/world.xml)

Raccourcis clavier :

Raccourci Fonction
Ctrl+N Créer une nouvelle carte
Ctrl+O Ouvrir une carte
Ctrl+S Enregistrer la carte
Ctrl+Shit+S Enregistrer la carte sous un autre nom de fichier
Ctrl+Q Quitter l'éditeur
U Annuler le dernier placement de tile
1 Ajouter une unité
2 Ajouter un décor
3 Ajouter un objet
D Déplacer une entité
S Supprimer une entité
Flèches directionnelles Se déplacer dans les zones

Positionner des tiles

Choisissez une tile dans le tileset en cliquant dessus (partie gauche de l'écran), puis cliquez dans la zone (partie droite) pour positionner la tile. Maintenez le bouton de la souris enfoncé en la déplaçant pour placer plusieurs tiles à la fois.
Les placements peuvent être annulés avec l'option Édition > Annuler tile.
Pour remplir toute la zone avec une tile, utilisez l'option Édition > Peindre avec la tile courante.

Déplacer une entité

Cliquez sur l'entité à déplacer (celle-ci se mettra à suivre le curseur), puis cliquez sur la nouvelle position de l'entité.

Supprimer une entité

Cliquez sur l'entité à supprimer.

Attention, ajouter des lignes ou des colonnes de zones dans une carte peut invalider les destinations des téléporteurs en raison d'un éventuel décalage de la position de la zone cible. Cela implique aussi bien les téléporteurs de la carte éditée, que les téléporteurs des autres cartes faisant référence à la carte éditée.

Fichier de configuration

L'éditeur utilise le fichier de configuration editor/config/config.txt pour définir l'emplacement des fichiers ressources.

Les valeurs suivantes doivent s'y trouver (syntaxe clef = valeur) :

  • map_path : répertoire par défaut des fichiers cartes
  • units : chemin du fichier XML des définitions des unités
  • decors : chemin du fichier XML des définitions des décors
  • tileset : chemin de l'image du tileset
  • musics : chemin de la liste des musiques du jeu
